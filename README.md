## Bookstore React-Redux

This is a React-Redux BookStore project.

## Requirements

- Node.js

- React

- React-DOM

- React-Create-App

- Redux

- npm

- CSS

- ES6

## Installation

Run `npm install`

## Available Scripts

- `npm start`

- `npm test`

- `npm run build`

- `npm run eject`
